# Heartbeat Configuration

## 🎯 Goal Alignment Check (every heartbeat)
- Is current work aligned with annual goal?
- Is time being spent on Track 1 or Track 2?
- Flag goal drift → alert + recovery suggestion

## 📊 Task Auto-Alignment (every heartbeat)
- Check active tasks against goals
- Flag misaligned or stale tasks
- Remind incomplete daily tasks

## 🔍 Self-Improvement Loop (weekly, Sunday)
- Review agent performance
- Identify wins, failures, missed opportunities
- Propose workflow improvements

## 📋 Audit Log (every heartbeat)
- Track external actions (emails, messages, uploads)
- Track internal changes (file edits, note creation)
- Alert on anomalies

## 📈 Performance Scorecard (weekly, Monday)
- Count outputs by track
- Analyze time allocation
- Check lead/pipeline movement
- Deliver weekly report

## 📦 Daily Backup (once daily, morning)
- Compress agent state files
- Store locally + cloud
- Keep 3-day rolling window

## 🧠 Memory Freshness (every heartbeat)
- Check MEMORY.md section dates
- Refresh stale sections
- Resolve pending items older than 7 days
